# 🎮 Open PS2 Loader (OPL) Repository Structure

This repository contains a complete **OPL (Open PS2 Loader)** directory layout, including applications, artwork, configurations, and themes.

It is designed to be deployed directly on a PlayStation 2 storage device (USB, HDD, or SMB).

---

## 📂 Repository Structure

```bash
OPL/
├── APPS/        # PSU packaged applications
├── ART/         # Game artwork and UI assets
├── CFG/         # Per-game configuration files
├── THM/         # UI themes
├── CD/          # CD-based games
├── DVD/         # DVD-based games
├── CHT/         # Cheat files
├── LNG/         # Language files
├── POPS/        # PS1 (POPS) support files
├── VMC/         # Virtual Memory Cards
└── ...
```

---

## 📁 Directory Details

### 📦 `APPS/` — PSU Applications

This directory contains applications packaged as **`.PSU` files**.

* PSU = PlayStation 2 save archive format
* Typically installed/extracted via **uLaunchELF (WLE)**
* Used to distribute homebrew tools in a memory card–compatible format

#### Included tools (examples):

* `APP_WLE-ISR-XF-MM.psu` → uLaunchELF (file manager)
* `DST_PS2IDENT.psu` → Hardware identification tool
* `DST_PS2TEMPS.psu` → Temperature monitoring
* `DST_HDDTESTER.psu` → HDD diagnostics
* `SYS_NTPS2.psu` → NTP time synchronization

> ⚠️ These files are **not directly executed by OPL**. They must be restored to a Memory Card or extracted.

---

### 🖼️ `ART/` — Artwork

Contains all graphical assets used by OPL.

Supported asset types include:

* `COV.png` → Cover image
* `COV2.png` → Alternate cover
* `BG_XX.png` → Background images
* `SCR_XX.png` → Screenshots
* `ICO.png` → Icon
* `LGO.png` → Logo
* `LAB.png` → Label

#### Naming convention

Files must match the **Game ID** or ELF name:

```bash
SCES_502.94_COV.png
SLUS_210.59_BG_00.png
HDDTESTER.ELF_COV.png
```

Strict naming is required for OPL to associate assets correctly.

---

### ⚙️ `CFG/` — Configuration Files

Per-game configuration files:

* Format: `<GAME_ID>.cfg`
* Automatically generated by OPL
* Can be manually edited for advanced tuning

Typical use cases:

* Compatibility modes
* GSM/video settings
* Per-game fixes

---

### 🎨 `THM/` — Themes

Contains full UI themes for OPL.

Each theme includes:

* Fonts (`.ttf`)
* UI graphics (`.png`)
* Sounds (`.adp`)
* Configuration (`conf_theme.cfg`)

Example:

```bash
THM/
└── thm_Pixel-Prime-Top.NoLogo/
```

Themes are selectable from within OPL.

---

### 💿 `CD/` and `DVD/`

Storage locations for PlayStation 2 games:

* `CD/` → CD-based titles
* `DVD/` → DVD-based titles

---

### 🎯 `CHT/`

Contains cheat files used by OPL.

---

### 🌐 `LNG/`

Language packs for OPL interface localization.

---

### 🕹️ `POPS/`

Files required for **PS1 game compatibility** via POPStarter.

---

### 💾 `VMC/` — Virtual Memory Cards

Stores virtual memory card files used by OPL.

---

## 🚀 Usage

1. Copy the `OPL` directory to your target device:

   * USB drive
   * Internal HDD
   * SMB network share

2. Launch OPL on your PlayStation 2.

3. Configure paths if required.

4. OPL will automatically detect:

   * Artwork
   * Configurations
   * Themes

---

## ⚠️ Important Notes

* File naming consistency is **critical**.
* Use standard **Game IDs** (e.g., `SCES_XXX.XX`, `SLUS_XXX.XX`).
* Avoid special characters in filenames.
* PSU files in `APPS/` require extraction before use.
